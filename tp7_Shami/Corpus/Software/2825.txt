   HTML5 ( HyperText Markup Language 5 ) est la dernière révision majeure
   du HTML ( format de données conçu pour représenter les pages web ).
   Cette version a été finalisée le 28 octobre 2014 . HTML5 spécifie deux
   syntaxes d'un modèle abstrait défini en termes de DOM : HTML5 et XHTML5
