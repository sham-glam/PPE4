   En informatique , un compilateur [1] est un programme qui transforme un
   code source en un code objet [2] . Généralement, le code source est
   écrit dans un langage de programmation (le langage source ), il est de
   haut niveau d' abstraction , et facilement compréhensible par l'humain.
   Le code objet est généralement écrit en langage de plus bas niveau
   (appelé langage cible ), par exemple un langage d'assemblage ou langage
   machine , afin de créer un programme exécutable par une machine.

   Compilateurs : principes, techniques et outils (en anglais Compilers:
   Principles, Techniques, and Tools ) est un ouvrage de référence en
   informatique dans le domaine de la compilation , écrit par Alfred V.
   Aho , Ravi Sethi et Jeffrey D. Ullman . Il traite de tous les aspects
   scientifiques et techniques de la construction de compilateurs.
