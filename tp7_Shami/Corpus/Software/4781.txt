   Windows Server 2003 est un système d'exploitation orienté serveur
   développé par Microsoft . Présenté le 24 avril 2003 comme le successeur
   de Windows 2000 Server , il est considéré par Microsoft comme étant la
   pierre angulaire de la ligne de produits serveurs professionnels
   Windows Server System . Une version évoluée intitulée Windows Server
   2003 R2 a été finalisée le 6 décembre 2005 . Son successeur, Windows
   Server 2008 , est sorti le 4 février 2008 . Le 14 juillet 2015 ,
   Microsoft cesse de le supporter, ce qui signifie l'arrêt de la
   publication de correctifs de sécurité, et encourage la migration vers
   Windows Server 2012 R2 [1] .
