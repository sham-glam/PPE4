   GitHub ( / ɡ ɪ t h ʌ b / , entreprise GitHub, Inc. ) est un service web
   d' hébergement et de gestion de développement de logiciels , utilisant
   le logiciel de gestion de versions Git . Ce site est développé en Ruby
   on Rails et Erlang par Chris Wanstrath, PJ Hyett et Tom Preston-Werner.
   GitHub propose des comptes professionnels payants, ainsi que des
   comptes gratuits pour les projets de logiciels libres .
